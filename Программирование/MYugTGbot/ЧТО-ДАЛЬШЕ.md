## 💡 Советы

### Inline клавиатуры лучше обычных
```typescript
// Вместо обычных кнопок (из flows.json):
keyboard: [['Заказы', 'Пользователи']]

// Используем inline:
inline_keyboard: [
  [{ text: '📦 Заказы', callback_data: 'orders' }],
  [{ text: '👥 Пользователи', callback_data: 'users' }]
]
```

### SQL запросы проще всего
Копируй SQL из flows.json как есть, они работают с ITM:
```typescript
const query = `
  SELECT O.ID, O.ITM_ORDERNUM, O.CLIENT
  FROM ORDERS O
  WHERE O.ORDER_STATUS > -3
`;
```
