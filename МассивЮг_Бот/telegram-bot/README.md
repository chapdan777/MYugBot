## Telegram-бот (Node.js) — миграция из Node-RED

### Обзор
Этот проект создает каркас контейнеризованного Telegram-бота на Node.js в соответствии с планом миграции из `dockerize-telegram-bot.md`. Включает инициализацию бота, маршрутизацию сообщений, управление состоянием в памяти, планировщики и Docker-конфигурацию.

### Требования
- Node.js 18+
- Docker (необязательно, для контейнерного запуска)

### Установка
1. Скопируйте `.env.example` в `.env` и заполните значения.
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запуск в режиме разработки:
   ```bash
   npm run dev
   ```
4. Запуск в продакшене:
   ```bash
   npm start
   ```

### Docker
Сборка и запуск через Compose:
```bash
cd docker
docker compose up --build -d
```
